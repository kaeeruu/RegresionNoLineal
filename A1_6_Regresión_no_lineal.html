1. Se importa la base de datos "A1.6 Tiempo de Entrega.csv" con la funci√≥n read_csv() de la librer√≠a pandas. Debido a que solamente hay dos variables: "Tiempo" y "Calificacion", se declaran x e y como cada una de las columnas.

Posteriormente, utilizando la librer√≠a matplotlib.pyplot, se genera una gr√°fica de dispersi√≥n para observar el comportamiento de los datos de manera visual.

La base de datos conten√≠a algunos elementos "raros". En la columna de calificaciones, algunas entradas eran caracteres, cuando deber√≠an ser n√∫meros y otras ten√≠an el t√©rmino NA la cu√°l se le√≠a como str al momento de intentar hacer la gr√°fica. Para resolver esto, se utiliz√≥ la funci√≥n to_numeric() con el par√°metro errors = "coerce" para convertir este tipo de entradas a tipo NaN y posteriormente eliminarlas con el comando dropna() ya que no tienen informaci√≥n que sirva para el an√°lisis de los datos.

Adicionalmente, se encontraron valores at√≠picos en la columna de Tiempo. En uno, el tiempo era de 5000 horas, lo cual no ten√≠a sentido ya que eso significar√≠a que el alumno entrego la actividad con aproximadamente 200 d√≠as de anticipaci√≥n, es decir 2 semestres antes. Es por esto que se excluyeron valores arriba de 336, lo que equivale a 2 semanas. Otro valor era negativo, lo cual podr√≠a significar una entrega tard√≠a, sin embargo, una entrega tard√≠a no ser√≠a √∫til para el prop√≥sito del an√°lisis por lo cual se eliminaron datos con Tiempo menor a 0.

In¬†[¬†]:
import pandas as pd
import matplotlib.pyplot as plt
df = pd.read_csv("A1.6 Tiempo de Entrega.csv")
x = "Tiempo"
y = "Calificacion"

df[x] = pd.to_numeric(df[x],errors='coerce')
df[y] = pd.to_numeric(df[y],errors='coerce')
df = df.dropna()

df = df[df[x] <= 336]
df = df[df[x] >= 0]
plt.figure(figsize=(8,8))
plt.scatter(df[x],df[y], alpha=0.5, edgecolors="k")

plt.xlabel(x)
plt.ylabel(y)
plt.title("Gr√°fica de Dispersi√≥n de {} vs {}".format(x,y))

plt.grid(True)
plt.show()

3. Primero se probar√° un modelo de regresi√≥n polinomial. Para esto se importa la funci√≥n PolynomialFeatures() de la librer√≠a sklearn.preprocessing y se importa tambi√©n la librer√≠a statsmodels.api.

Se modifican los datos para adaptarse a un modelo polinomial utilizando las funciones array.reshape(-1,1) y se establece el par√°metro degree = 4. Apartir de estos datos modificados se crean 2 dataframes nuevos sobre los cuales se crear√° el modelo.

Utilizando la funci√≥n OLS() de statsmodels.api se entrena el modelo con el comando .fit() y se imprime un resumen de este. Se utiliza este modelo con los datos de validaci√≥n para observar los resultados usando el comando predict().

De igual manera utilizando la librer√≠a matplotlib.pyplot se genera una gr√°fica de dispersi√≥n donde se muestran ambos valores reales de "Calificacion" sobre "Tiempo" y los valores estimados de "Calificacion" utilizando el modelo.

In¬†[¬†]:
from sklearn.preprocessing import PolynomialFeatures
import statsmodels.api as sm

x_tmp_train = x_train.array.reshape(-1,1)
x_tmp_test = x_test.array.reshape(-1,1)
polyq_train = PolynomialFeatures(degree=4).fit(x_tmp_train)
polyq_test = PolynomialFeatures(degree=4).fit(x_tmp_test)
xq_train = pd.DataFrame(polyq_train.transform(x_tmp_train))
xq_test = pd.DataFrame(polyq_test.transform(x_tmp_test))
mq = sm.OLS(list(y_train),xq_train).fit()
print(mq.summary())
plt.scatter(x_test,y_test, color="lightblue",alpha = 0.5,label="Datos Reales")
yhat_mq = mq.predict(xq_test)
plt.scatter(x_test,yhat_mq, color = "red",alpha = 0.5, label="Regresi√≥n Polinomial")
plt.xlabel("Tiempo")
plt.ylabel("Calificaci√≥n")
plt.title("Regresi√≥n Polinomial")
plt.legend()
plt.grid(True)
plt.show()
